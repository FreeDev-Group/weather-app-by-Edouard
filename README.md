# Frontend Weather App - Real-time Forecast

This is a **frontend project**: a real-time weather forecast application designed to provide current weather conditions, hourly and daily forecasts, and interactive features across all devices. It demonstrates responsive design, API integration, and smooth UI animations.

## Overview

### The project

Users should be able to:

- View current weather conditions for any city.
- Switch between Celsius and Fahrenheit units.
- See hourly and daily forecasts with clear visual representation.
- Enjoy smooth hover effects and subtle animations.
- Experience a fully responsive layout across desktop, tablet, and mobile devices.

## My process

### Built with

- **HTML5** - Semantic markup and proper heading hierarchy  
- **CSS3** - Flexbox, Grid, gradients, hover effects, and animations  
- **Vanilla JavaScript** - DOM manipulation, event handling, and API integration  
- **OpenWeatherMap API** - Fetch and display real-time weather data  
- **Mobile-first workflow** - Fully responsive design  
- **Custom animations** - Hover effects and floating gradients  
- **CSS Grid & Flexbox** - Modern layout techniques  

### What I learned

Through this project, I enhanced my skills in:

Responsive Design: Ensured the layout works on desktop, tablet, and mobile.
JavaScript DOM Manipulation: Fetched and displayed API data dynamically, updated UI based on user actions.
CSS Animations & UI Effects: Smooth hover effects and floating gradient animations.
API Integration: Worked with OpenWeatherMap for real-time weather updates.
Accessibility & Usability: Improved interactive elements for keyboard and mobile users.
Performance Optimization: Minimized layout shifts and optimized grids for speed.
Features implemented

### Enhanced Features Beyond Design:

* Search bar with autocomplete suggestions
* Current weather display with dynamic background gradient
* Hourly forecast cards with hover effects
* Daily forecast cards with min/max temperatures
* Unit toggle (Celsius / Fahrenheit)
* Fully responsive layout for desktop, tablet, and mobile
* Smooth hover and floating animations
* Dropdown menus for unit selection and day selection
* Accessible design for keyboard and mobile users
